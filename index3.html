Олимпиадная задача:

Коля переставляет карточки с цифрами 9, 4, 1, 8, 0 относительно друг друга и записывает получающиеся четырехзначные числа. Какое наименьшее количество чесел из полученных может выбрать Коля, чтобы среди них гарантированно оказалось хотя бы одно нечетное?Ответ пояснить.
Чтобы открыть решение - отправьте код подтверждения "Captcha code"

Решение: Кол-во комбинаций из пяти чисел равно 5!=1*2*3*4*5=24*5=120. То есть 120 разных четырехзначных комбинаций можно составить из пяти чисел. Для выполнения условия нечетности получившегося числа необходимо, чтобы на последнем месте стояла либо 9 либо 1, итого два числа из пяти (согласно предложенного набора). 120*2/5=48, столько существует нечетных вариантов из 120. Значит чтобы гарантированно оказалось хотя бы одно четное, необходимо взять все четные и одно нечетное: (120-48)+1=73
